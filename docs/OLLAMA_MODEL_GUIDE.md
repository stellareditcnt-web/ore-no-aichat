# Ollama モデル選定 & 導入ガイド

このアプリ（俺のAIChat）で利用できる、主要なAIモデルの特性とインストールコマンドをまとめました。

## 1. 目的別のおすすめモデル

| カテゴリ | モデル名 | 特性・メリット | インストールコマンド |
| :--- | :--- | :--- | :--- |
| **無修正 (最強)** | `dolphin-llama3` | **セーフティが完全解除**されており、過激な描写も拒まない。現状のデフォルト。 | `ollama pull dolphin-llama3` |
| **日本語＋無修正** | `mistral-nemo` | 12Bと少し大きめだが、**日本語が非常に自然**。セーフティも比較的緩いバランス型。 | `ollama pull mistral-nemo` |
| **日本語 (最高精度)** | `gemma2` | Google製。**日本語の美しさはNo.1**。ただし少し真面目（セーフティが強め）。 | `ollama pull gemma2` |
| **日本語 (定番)** | `elyza:8b` | Llama 3を日本企業が微調整。**日本の文脈に強い**。物語の「情緒」を出すのに最適。 | `ollama pull elyza:8b` |
| **軽量・高速** | `phi3:mini` | Microsoft製。非常に軽量で古いMacでも爆速。指示への忠実度が高い。 | `ollama pull phi3:mini` |

## 2. モデルの切り替え方法

新しいモデルを試す手順は以下の通りです。

1. **ターミナルでダウンロード**:
   上記のインストールコマンドを実行して完了を待ちます。
2. **設定ファイル (`.env`) を書き換える**:
   `LOCAL_AI_MODEL=` の後ろの部分を、使いたいモデル名に書き換えて保存します。
   例: `LOCAL_AI_MODEL=mistral-nemo`
3. **アプリで確認**:
   ブラウザをリロード（または起動スクリプトを実行）して、新しいモデルでの反応を確かめてください。

## 3. 困った時のコマンド集

- **今入っているモデルの一覧を見る**:
  ```bash
  ollama list
  ```
- **不要になったモデルを消す**:
  ```bash
  ollama rm モデル名
  ```
- **Ollamaが動いているか確認**:
  ```bash
  ollama ps
  ```
## 4. モデルの保存場所（Mac）

Ollamaでダウンロードしたモデルファイルは、デフォルトで以下の場所に保存されています。

- **場所**: `/Users/あなたのユーザー名/.ollama/models`
  - ※ `.ollama` は隠しフォルダです。

**Finderで直接開くコマンド**:
ターミナルに以下を貼り付けてエンターを押すと、モデルが保存されているフォルダがパッと開きます。
```bash
open ~/.ollama/models
```

> [!CAUTION]
> モデルファイルは1つ数GBと非常に大きいです。Macの空き容量が少なくなってきたら、使わなくなったモデルを `ollama rm モデル名` で削除して整理することをおすすめします。

> [!TIP]
> **「日本語が不自然だな」と感じたら...**
> 無修正の `dolphin-llama3` ではなく、一度 `mistral-nemo` や `gemma2` を試してみてください。日本語の精度が上がると、物語の没入感が格段にアップします。
